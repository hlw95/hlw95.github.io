3 Java 源文件的命名规则

  • 通常情况下，Java 程序源文件的文件名可以任意。
  • 但如果其中定义了一个 public 类，则该源文件的文件名必须与该 public 类的类名相同。
  • 一个Java 源文件可包含多个类定义，但最多只能包含一个public类定义。
